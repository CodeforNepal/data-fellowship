# Data Fellowship landing page

## Installing Dependencies

```bash
yarn
```

## Development

### Prerequisits

- Create `_contents` folder
- Copy contents from `demo_data` folder to `_contents`

### Start Script

```bash
yarn dev
```

## Environment

Env File

- .env.development.local

```bash
NEXT_PUBLIC_FORM_URL=
```

## Production

```bash
yarn build
```
